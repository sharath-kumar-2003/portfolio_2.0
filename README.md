## 🌗 Overview

A premium, immersive personal portfolio built around a **dual-theme "Sun & Moon"** concept — **Sacred Sunlight** and **All-Embracing Moonlight** — with seamless animated transitions between both modes. Designed as a cinematic, editorial-grade single-page experience that reflects a deep focus on clarity, precision, and intelligent systems.

> *"I build systems that see through complexity. Most impactful intelligence begins in the quiet."*

---

## ✨ Features

| Feature | Description |
|---|---|
| 🌞 / 🌙 **Dual Theme** | Toggle between Sacred Sunlight and All-Embracing Moonlight with a smooth flash transition |
| 🎠 **3D Project Carousel** | CSS-perspective horizontal scroll carousel with magnetic card interactions |
| 🏗️ **Bento Grid Layout** | Editorial-style bento grid for the About section with reveal animations |
| 🌀 **Ambient Background** | Procedural conic rays, radial glows, floating particles, and noise texture |
| ⚡ **Scroll Animations** | Intersection Observer-based reveal animations with staggered delays |
| 📈 **Animated Stats** | Counting-up number animations triggered on scroll |
| 🧲 **Magnetic Buttons** | Cursor-tracking magnetic hover effect on CTA buttons |
| 🔢 **Tech Marquee** | Auto-scrolling tech stack ticker |
| 📱 **Responsive** | Mobile menu, touch-swipe carousel, and fully responsive layout |
| 💾 **Theme Persistence** | Theme choice saved in `localStorage` across sessions |

## 🎨 Design System

The portfolio uses a **CSS custom property architecture** with two complete theme palettes:

- **☀️ Sunlight** — Warm golds, amber glows, cream surfaces, long conic shadow rays
- **🌙 Moonlight** — Deep navy, silver highlights, cool indigo accents, star layers

Transitions between themes use a full-screen flash overlay for a cinematic feel.
